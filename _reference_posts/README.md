# Reference Posts

This directory contains example blog posts from the al-folio theme that are kept for reference purposes. These posts demonstrate various formatting features and capabilities of the theme.

These posts are **not** processed by Jekyll and will **not** appear on your website. They are stored here for your reference when creating new blog posts.

## Available Examples

- **Formatting**: Links, images, code, math, tables, blockquotes
- **Media**: Videos, audios, image galleries
- **Interactive**: Charts (Chart.js, ECharts, Vega-Lite, Plotly), maps (GeoJSON), diagrams (Mermaid, TikZ, Typograms)
- **Features**: Comments (Disqus, Giscus), table of contents, Jupyter notebooks, citations, tabs, code diffs
- **Layouts**: Distill-style posts, redirects

To use any of these examples, copy the relevant post to the `_posts` directory and modify it for your needs.
